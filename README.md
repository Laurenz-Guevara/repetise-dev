## Getting Started

If you would like to build this app locally for yourself just follow the instructions below.

### Node Version

v14.18.0

### Installation

1. Clone the repo

    ```bash
    git clone https://github.com/Laurenz-Guevara/repetise-dev
    ```

2. Install Packages

    ```bash
    npm install
    ```
    
3. Run a Build

    ```bash
    npm run build
    ```

4. Start the App

    ```bash
    npm run dev
    ```

### Deploy Repetise

1. Run a Build

    ```bash
    npm run build
    ```

2. Deploy on firebase

    ```bash
    firebase deploy
    ```
